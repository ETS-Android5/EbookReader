# EbookReader

电子书阅读器

The EbookReader is an Android application for reading books in HTML, SVG, PDF, EPUB, CBZ, MOBI, TXT, AZW, AZW3 formats on Android devices.

-----------------------

2021.12.11 增加html, svg, mobi, azw, azw3, azw4格式电子书支持。
